

# C语言学习

这是一个C语言学习项目，包含了多个CS50课程风格的示例程序和练习代码，适合初学者学习和实践C语言基础。

## 项目结构

### 基础入门

| 文件 | 说明 |
|------|------|
| `hi.c` | 基础输出示例 |
| `numbers.c` | 数字类型演示 |
| `float.c` | 浮点数类型 |
| `string.c` | 字符串处理 |
| `block1.c` / `block2.c` | 代码块示例 |

### 控制流

| 文件 | 说明 |
|------|------|
| `for.c` | for循环语句 |
| `switch1.c` | switch分支语句 |
| `mario2.c` | 马里奥风格输出 |

### 函数

| 文件 | 说明 |
|------|------|
| `cough0.c` | 无参函数 |
| `cough1.c` | 函数定义与调用 |
| `cough3.c` | 带参数的函数 |

### 命令行参数

| 文件 | 说明 |
|------|------|
| `argv.c` | 命令行参数处理 |
| `exit.c` | 程序退出处理 |
| `agree.c` | 用户交互示例 |

### 高级主题

| 文件 | 说明 |
|------|------|
| `scores.c` | 数组与指针、函数返回 |
| `phonebook.c` | 结构体使用 |
| `overflow.c` | 整数溢出演示 |
| `uppercase.c` | 字符转换 |
| `names.c` | 名字处理 |

## 编译运行

```bash
# 编译单个源文件
gcc -o output/names names.c

# 运行生成的可执行文件
./output/names
```

## 依赖

- GCC 编译器
- Linux/macOS/Windows (需配置GCC环境)
