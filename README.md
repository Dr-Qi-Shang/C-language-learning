

# C语言学习

这是一个C语言学习项目，包含了多个CS50课程风格的示例程序和练习代码，适合初学者学习和实践C语言基础。项目还包含了一些LeetCode算法题的解题代码，帮助学习者提升编程能力。

## 项目结构

### 基础入门

| 文件 | 说明 |
|------|------|
| `hi.c` | 基础输出示例，演示Hello World输出 |
| `numbers.c` | 数字类型演示，展示整数的使用 |
| `float.c` | 浮点数类型的演示 |
| `string.c` | 字符串处理基础 |
| `block1.c` / `block2.c` | 代码块示例，演示作用域概念 |

### 控制流

| 文件 | 说明 |
|------|------|
| `for.c` | for循环语句的用法 |
| `switch1.c` | switch分支语句演示 |
| `mario2.c` | 马里奥风格输出，循环练习 |

### 函数

| 文件 | 说明 |
|------|------|
| `cough0.c` | 无参函数定义与调用 |
| `cough1.c` | 函数基础定义 |
| `cough3.c` | 带参数的函数 |

### 命令行参数

| 文件 | 说明 |
|------|------|
| `argv.c` | 命令行参数处理 |
| `exit.c` | 程序退出状态处理 |
| `agree.c` | 用户交互示例 |

### 高级主题

| 文件 | 说明 |
|------|------|
| `scores.c` | 数组与指针、函数返回值的处理 |
| `phonebook.c` | 结构体使用实战 |
| `overflow.c` | 整数溢出演示 |
| `uppercase.c` | 字符转换操作 |
| `names.c` | 名字处理示例 |

### 递归与迭代

| 文件 | 说明 |
|------|------|
| `recursion(di gui).c` | 递归函数练习 |
| `iteration(Die Dai).c` | 迭代循环练习 |

### LeetCode 算法题

| 文件 | 说明 |
|------|------|
| `136.c` | 只出现一次的数字 |
| `704.c` | 二分查找 |

## 编译运行

```bash
# 编译单个源文件
gcc -o output/names names.c

# 运行生成的可执行文件
./output/names
```

## 依赖

- GCC 编译器
- Linux / macOS / Windows (需配置GCC环境)
